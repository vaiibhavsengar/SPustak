# 📒 Simple Notes App

**S-Pustak** is a simple and intuitive notes application built using **HTML, CSS, and JavaScript**. It helps users capture ideas, manage notes, and keep track of important tasks in an organized manner.

## ✨ Features

* 📝 Create and save notes
* ✏️ Edit existing notes
* 🗑️ Delete unwanted notes
* ✅ Add and manage tasks
* ⚡ Fast and lightweight
* 📱 Responsive and user-friendly design
* 🛠️ Built with vanilla JavaScript

## 🔄 How It Works

1. 🚀 Launch the application.
2. 📝 Create a new note or task.
3. ✏️ Update information whenever needed.
4. 🗑️ Remove notes that are no longer required.
5. 📚 Keep your thoughts and tasks organized.
